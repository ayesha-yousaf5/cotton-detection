🌿 Cotton Disease Classifier

An AI-powered web application that detects diseases in cotton plants using deep learning. Users can upload an image or capture one via webcam to instantly identify the disease and receive treatment recommendations.

🚀 Features 
📸 Upload cotton leaf/stem images
📷 Real-time webcam capture <br>
🤖 Deep Learning-based classification (CNN model) <br>
📊 Prediction confidence score <br> 
🩺 Treatment suggestions for each disease <br>
🎨 Clean and interactive UI built with Streamlit <br>
🌍 Bilingual support (English + Urdu disease names) <br>
🧠 Supported Disease Classes <br>

The model can classify the following cotton plant conditions:

🌿 Aphids (افڈز) <br>  
🐛 Army Worm (آرمی کیڑا) <br> 
🦠 Bacterial Blight (بیکٹیریل جھلساؤ) <br> 
🍃 Powdery Mildew (سفوفی پھپھوند) <br> 
🎯 Target Spot (نشانی دھبہ)<br> 
🌼 Healthy Plant <br>
